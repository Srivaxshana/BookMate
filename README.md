BookMate is a full-stack bookstore management app to browse, search, and manage books. Admins can add, update, and delete books, while users can view book details. The app uses a frontend, backend, and database with Dockerized setup and CI/CD support.

## Features
- Browse and search books
- Add, update, delete books (admin)
- View book details
- Docker & CI/CD support

## Tech Stack
- Frontend: React, HTML, CSS, JavaScript
- Backend: Java Spring Boot (or Node.js)
- Database: MySQL / PostgreSQL
- DevOps: Docker, Jenkins
